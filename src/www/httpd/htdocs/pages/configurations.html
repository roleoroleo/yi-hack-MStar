Configurations

In this page you can enable and disable services, change the camera hostname, disable the cloud features and more.

General

Hostname Camera's hostname on the network. Leave blank to generate automatically.
Timezone Timezone. Get your timezone string from this page and copy here.
Disable Cloud Disable the Yi App and all the Cloud features (aka private mode).
Recording without Cloud Enable recording on the SD card even with the Cloud features disabled. (It takes effect only if the "Disable Cloud" option is enabled).
RTSP Enable RTSP Server.
RTSP alternative server Use alternative RTSP server instead of the live555 based one. This version doesn't support aac audio and noise reduction.
RTSP Stream
Select stream resolution.
RTSP Audio
Enable/disable audio for RTSP stream.
Audio Noise Reduction Level
Select noise reduction level for audio stream (only with pcm, alaw or ulaw). Select "0" to disable noise reduction and save CPU time.
ONVIF Enable ONVIF Server.
ONVIF configuration
Enable/disable time OSD Enable/disable time OSD in the upper left corner.
It works only if the cam doesn't show osd natively.
Warning: this option is not fully compatible with the app. If you use the app you will notice a shift in the timeline.

Snapshot

Snapshot Enable snapshot feature.
If you want to use this feature, enable swap file to avoid memory problems.
Snapshot for recorded video Create snapshot for recorded video (CPU expensive).
Force low resolution snapshot Take low resolution snapshots only (useful for avoiding memory problems).
Time-lapse Enable time-lapse snapshots.
Time-lapse interval Interval between frames (minutes: 1-6, 10, 15, 20, 30, 60, 120, 180, 240, 360, 1440).
If you use 1440, please specify offset from midnight with the syntax 1440+offset (example to take snapshot at noon: 1440+720).
Time-lapse video creation interval Interval between video creation: too difficult to make a gui, use crontab syntax.
For example: "0 0 * * *" to make a video every day at midnight.
Disabled if you leave the field empty or if you enable "Time-lapse ftp push".
Time-lapse video Go to this page to see time-lapse video
Time-lapse ftp push Send time-lapse snapshots to the ftp server (see Events page for ftp options) and remove local file.
Overwrite ftp destination file (use same name) Send time-lapse snapshots to the ftp server using the same name.

Advanced

SSH Enable or disable the SSH server.
FTP Enable or disable the FTP server.
Legacy FTP Switch to the legacy BusyBox FTP server.
NOTE: If you are trying to disable Pure-FTPd as mentioned in this issue you can enable this option instead of manually editing the system.sh file.
Telnet Enable or disable the Telnet server.
WARNING: Telnet is NOT a secure protocol and should be avoided. Enable only if it's really necessary.
NTPD Enable or disable the Network Time Protocol Daemon. This is required to sync the time on the camera if the cloud features are disabled.
NTP Server NTP server name.
HTTPD Enable or disable the Web Server.
WARNING: If disabled, you won't have access to this page.
Proxychains-ng Proxychains-ng is a way to bypass the region restrictions imposed by Yi when using a Chinese camera outside China. See the GitHub page for more info on how to configure it.

TCP ports

RTSP Port TCP port used by RTSP Server (default 554).
HTTPD Port TCP port used by HTTPD Server (default 80).

Authentication

Username Username to access RTSP, ONVIF and HTTP (default empty).
Password Password to access RTSP, ONVIF and HTTP (default empty).
Allowed special chars: \ | ! " $ % & / ( ) ' ? ^ [ ] @ < > , ; . : - _ *
Root Password Password for SSH, Telnet and FTP root user (default empty).
Allowed special chars: \ | ! " $ % & / ( ) ' ? ^ [ ] @ < > , ; . : - _ *
